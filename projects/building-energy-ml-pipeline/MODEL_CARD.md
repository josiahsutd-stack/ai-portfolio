# Model Card: Building Energy Prediction Demo

## Intended Use

Estimate annual building energy intensity from synthetic building features for portfolio demonstration and local experimentation.

## Data

The dataset is generated by `scripts/generate_sample_data.py`. It is synthetic, public-style data and must not be treated as measured building performance.

## Model

Random forest regressor with one-hot encoded categorical features and standardized numeric features.

## Metrics

The dashboard reports mean absolute error and R2 on a holdout split generated from the synthetic dataset.

## Ethical and Practical Limitations

- Not calibrated to any real jurisdiction, climate file, utility bill, or simulation engine.
- Should not be used for compliance, investment, or engineering decisions.
- Real deployment would require data provenance, uncertainty estimates, and expert review.
