Embodied AI

Language, state, action, and visible failure

The simulator maps task instructions and structured construction grids to closed-loop actions. It compares A* planning with a fitted behavior-cloning policy, then measures how an explicit action filter changes unsafe attempts, interventions, and task completion.

Policy
Random-forest action classifier fitted from simulator expert trajectories
Holdout
24 disjoint procedural scenarios with raw and filtered rollouts
Limit
Structured 2D state only; no perception, ROS, physics, hardware, or physical-safety validation
