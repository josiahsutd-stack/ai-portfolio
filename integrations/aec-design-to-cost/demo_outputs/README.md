# Generated Outputs

These files are deterministic outputs from:

```bash
python integrations/aec-design-to-cost/run_workflow.py
```

All inputs and rates are synthetic.

| File | Contents |
| --- | --- |
| `workflow_trace.json` | Full source map, approved requirement handoff, scenario, selected massing assessment, schematic plan, takeoff, synthetic estimate, review gates, and limitations |
| `workflow_summary.md` | Compact result summary for human inspection |
| `workflow_trace.svg` | Diagram rendered from values in the JSON trace |

The output is integration evidence only. It is not a design, compliance determination, professional cost estimate, or tender recommendation.
