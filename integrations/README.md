# Cross-Project Integration Evidence

This directory contains executable contracts between selected projects. These integrations are kept separate from `projects/` because they do not represent additional portfolio projects or new model claims.

| Integration | Connected systems | Evidence |
| --- | --- | --- |
| [AEC design-to-cost handoff](aec-design-to-cost/README.md) | Specification Copilot, Massing Explorer, and QS Workbench | Approved-requirement gating, field-level source records, deterministic geometry handoff, schematic takeoff, rejection tests, and generated trace |

All bundled inputs and rates are synthetic. The integration records software behavior only; it is not evidence of professional design, compliance, cost advice, or project delivery.
