# Generated Evidence

These files are regenerated by `python experiments/qs-takeoff-tender-analysis/evaluate_qs.py` from synthetic fixtures:

- `evaluation_summary.json` and `evaluation_report.md`: metric evidence and per-case results.
- `failure_analysis.md`: fixture mismatches and fail-closed behavior.
- `sample_takeoff.csv` and `sample_estimate.json`: auditable quantity and cost records.
- `sample_tender_analysis.json`: line-level tender exceptions.
- `sample_plan.svg`, `cost_breakdown.svg`, and `tender_comparison.svg`: visual explanations generated from the same calculated records.

They are demonstration artifacts, not professional QS deliverables or live commercial data.
