# Demo Recording Guide

Screenshots and video evidence should come from actual local runs.

## Suggested Recording

1. Start with the root README and show the three flagship projects.
2. Run `python scripts/generate_review_artifacts.py`.
3. Run `streamlit run projects/aec-code-compliance-rag/app.py`.
4. Ask one answerable AEC question and one no-evidence question.
5. Show the generated `demo_outputs/` files.
6. Briefly open the agent trace and MLOps model eval report.

## Claims To Avoid

- No production deployment claim.
- No real customer claim.
- No legal/code/compliance sign-off claim.
- No robot hardware claim.
- No real VLM reasoning claim unless a real model/provider is demonstrated.
