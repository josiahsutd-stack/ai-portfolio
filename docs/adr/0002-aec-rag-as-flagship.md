# ADR 0002: AEC RAG As Flagship

## Context

The candidate's differentiator is applied AI for built-environment workflows.

## Decision

Make AEC Code Compliance RAG the primary review project.

## Consequences

The repo has a clearer hiring signal and avoids presenting all projects as equally deep.

## Alternatives Considered

Agents, VLM, and MLOps were considered as co-flagships but are better as supporting evidence.

## Production Change

A real product would require live/public code sources, jurisdiction metadata, expert review, and legal boundaries.
