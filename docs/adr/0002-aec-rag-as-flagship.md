# ADR 0002: AEC RAG As Flagship

## Context

My strongest domain advantage is applying AI to built-environment workflows.

## Decision

Use AEC Code Compliance RAG as the portfolio's primary project.

## Consequences

The repository presents one deeply evaluated project instead of implying that every experiment has equal depth.

## Alternatives Considered

Agents, VLM, and MLOps were considered as co-flagships but are better as supporting evidence.

## Production Change

A real product would require live/public code sources, jurisdiction metadata, expert review, and legal boundaries.
