# Recruiter Project Snippets

## Short Intro

I am building an applied and embodied AI portfolio around AI for the built environment, combining my AI, architecture, and embodied AI background with runnable projects in RAG, computer vision workflows, BIM QA, construction robotics, energy prediction, and design automation.

## AEC Code Compliance RAG

I built a local RAG assistant that answers questions over synthetic AEC guidance documents using chunking, retrieval, source-grounded answer generation, and citation display. The project demonstrates LLM engineering for document-heavy architecture and construction workflows.

## Construction Progress CV

I built a construction progress tracking demo that classifies synthetic site-image metadata into project stages and generates progress reports through a dashboard and API. It demonstrates the product workflow around CV systems even without private site-image datasets.

## BIM Issue Detection Agent

I built a BIM QA assistant that parses mock room schedules, flags coordination issues such as duplicate IDs, area mismatches, missing fire notes, accessibility gaps, and unresolved comments, then produces an explainable issue report.

## Building Energy ML

I built a scikit-learn pipeline that predicts synthetic building energy intensity from architectural and operational features, including preprocessing, evaluation metrics, FastAPI deployment, a Streamlit dashboard, and a model card.

## Spatial Design Recommender

I built an explainable recommendation engine for spatial design scenarios, scoring daylight, circulation, room sizing, and zoning constraints and returning prioritized design actions.

## Construction Robot Task Planner

I built an embodied AI planning demo for construction robots that routes tasks across a synthetic site map while accounting for obstacles, restricted zones, slow zones, payload, and battery margin.

## Site Robot Safety Monitor

I built a construction robot safety-monitoring demo that analyzes synthetic robot telemetry and flags worker proximity, obstacle clearance, restricted-zone speed, payload stability, and emergency-stop events.
